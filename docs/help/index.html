DJBrain - Helpbrowser

About

This piece of software is designed to let a DJ get his job done more easily. You can easily manage your collection of Vinyls, Cds, Mp3s and save and visualize possible transitions between these tracks. The intention of this software is to let you plan your sets, and keep cool transitions in mind while playing around with music. With DJBrain your live set will never run out into dead ends and you will always have a track that perfectly fits into your set!

Import / Export

DJBrain can export its database (or the tracks only) to an xml file. This file can be imported into another (or new) Database.

Glossary

  • Track -
  • Current Track - If you select a track (i.e. doubleclicking it in the Tracklist) it becomes the current track and is shown in the Currenttrack view
  • Suggestion - For each track you can define its suggestions. A suggestion is a track which, you mean, fits to to the current track. The suggestions for the track shown in Currenttrack are shown in the Suggestionlist.
